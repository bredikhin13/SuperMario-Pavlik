Команда: Смотров, Швырева, Бредихин, Соколов.


Игра на Java Марио Платформер.


Персонаж (имеет 3 жизни).
Персонаж может:
1.Ходить.
2.Прыгать.

Враги - 2.
Враги могут:
1. Перемещаться между стенами.
2. Стрелять.


3 уровня. Сложность уровня зависит от количества врагов.
1 уровень - враг No 1.
2 уровень - враг No 2.
3 уровень - враг No 1, 2.
Уровень пройден, если персонаж дошел до конца и собрал определенное количество бонусов.


Управление игрой – с клавиатуры.
